Bookmarklet Installer

URL Cleaner

Install a bookmarklet that aggressively strips a page URL down to its essentials. It removes hashes, normalizes the domain, and drops nearly all query parameters by default, while keeping only a tiny allowlist for a few sites where parameters are usually required.

Install

Drag this button to your bookmarks bar:

URL Cleaner

If dragging does not work, copy the code and paste it into a new bookmark’s URL field.

Some browsers hide the bookmarks bar by default. In many browsers, Ctrl+Shift+B toggles it.

How to use it

  1. Install the bookmarklet.
  2. Go to any page with an ugly URL.
  3. Click URL Cleaner in your bookmarks bar.
  4. The cleaned URL is copied to your clipboard and also shown in a popup.
  5. It is intentionally aggressive and may remove parameters some sites use for state or special views.
